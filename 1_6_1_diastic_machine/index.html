Diastic Machine

This page takes inspiration from Jackson Mac Low's Diastic Technique.

A simple explanation, if possible, of the technique, requires a lengthy text and a seed phrase like this:

Base text:

Then we need a seed phrase, we'll take an easy one: "cat".

Now we read the base text, and find the first word starting with 'c', 'coals'.
Then we start over, from the first word after 'coals', and find the first word with 'a' as second letter, 'lad'.
And start again, after 'lad', with 't' as third letter, altogether.

And now we have our "poem": coals lad altogether.

Can be done like this, the good old fashioned way, but we have a computer, so why not use that.

You can supply your own base text, dropping it in the page, or use the one we have supplied, chapter one of Alice's Adventures in Wonderland.

You also have to provide a seed phrase, by writing is in the input field.
